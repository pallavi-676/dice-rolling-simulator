# 🎲 Dice Rolling Simulator

A simple and interactive command-line Dice Rolling Simulator built using Python. This project demonstrates the use of random number generation, loops, and ASCII-based UI rendering.

---

## 🚀 Features

- Roll one or multiple dice
- Random number generation using Python
- ASCII-based visual dice display
- Displays total and individual dice values
- User-friendly loop for repeated gameplay

---

## 🛠️ Tech Stack

- Python 3

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/pallavi-676/dice-rolling-simulator.git

2. Navigate to the folder:
   cd dice-rolling-simulator

3. Run the program:
   python dice_simulator.py
